INSTALLATION

    - Mettre le projet sur votre Bureau ou dans un endroit où vous allez le retrouver facilement.
    - Suivez les étapes de CONFIGURATION
    - Le lancement de l'application se fait par la classe HelloApplication.

CONFIGURATION

    - Lancez votre serveur WAMP et rendez vous sur la page d'accueil de phpMyAdmin.
    - Sélectionnez l'option "Importer" dans la barre de navigation de la page (milieu en haut normalement)
    - Téléversez le fichier "matable.sql" (qui est au même endroit que ce README) avec "Chercher un fichier".
    - Allez en bas de la page internet phpMyAdmin après avoir téléversé le fichier et cliquez sur "Importer".
